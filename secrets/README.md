# Secrets

Encrypted secrets managed by [agenix-rekey](https://github.com/oddlama/agenix-rekey).

## Structure

| File | Encrypted For | Purpose |
|------|---------------|---------|
| `*.age` | TPM only | Master copies (you edit these) |
| `../.secrets/rekeyed/openclaw/*.age` | Host only | Auto-generated by `agenix rekey` |

## Commands

```bash
# Create/edit a secret
agenix edit secrets/foo.age

# Rekey all secrets for hosts
agenix rekey

# Don't forget to commit rekeyed secrets
git add .secrets/
```

## Secret Formats

**hetzner-api.age**: Raw token
```
abc123...
```

**anthropic-api.age**: Environment variable
```
ANTHROPIC_API_KEY=sk-ant-xxxxx
```

**telegram-creds.age**: Environment variable
```
TELEGRAM_BOT_TOKEN=123456789:ABCdef...
```

**telegram-user-id.age**: Raw ID
```
1234567890
```

**gateway-token.age**: Environment variable
```
OPENCLAW_GATEWAY_TOKEN=abc123...
```
